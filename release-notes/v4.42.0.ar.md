# حساباتي 4.42.0

### Added
- الإعدادات: اسم جهاز اختياري (عربي أو إنجليزي) يُضمَّن في أسماء ملفات النسخ الاحتياطي وملف JSON — لتبقى نسخ كل جهاز منفصلة في مجلد مشترك (مثل Dropbox).
- نموذج الحساب: رابط الموقع يقبل النطاق بدون `https://` — التطبيق يضيف البروتوكول تلقائياً عند الحفظ.

### Changed
- أسماء ملفات النسخ التلقائي: `ishtarkati-latest-{device}.json` و`ishtarkati-auto-{device}-{timestamp}.json` — الحذف التلقائي يمس نسخ هذا الجهاز فقط.
