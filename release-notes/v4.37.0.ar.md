# حساباتي 4.37.0

### Added
- قائمة الحسابات: فلتر نوع المنصة (موقع / برنامج / الاثنين) وفلتر «ناقصة البيانات» للحسابات بدون بريد أو هاتف أو مستخدم؛ شريط يعرض العدد.
- تثبيت الحسابات (★) — تظهر في أعلى القائمة وفي الرئيسية تحت «حسابات مثبّتة».
- «يحتاج انتباهك»: تجارب تنتهي خلال 7 أيام؛ البطاقات المنتهية تعرض الحسابات المربوطة بها.
- صفحة التفاصيل: دمج حسابات مشابهة مكررة (نقل الدفعات ودمج البيانات).
