# Ishtarkati 4.19.0

### Added
- **شريط النظام (Tray)**: إغلاق النافذة يُبقي التطبيق يعمل في الخلفية مع قائمة إظهار / إنهاء.
- **تذكيرات في الخلفية**: إشعارات للمستحق اليوم وملخص للاشتراكات القادمة (حتى بدون فتح النافذة).
- **نسخ احتياطي تلقائي**: مجلد وفترة بالأيام من الإعدادات → نسخ وتصدير.
- **لوحة أوامر** (`Ctrl+K`): انتقال سريع بين الصفحات والبحث في الحسابات.
- **اختصارات لوحة المفاتيح**: `n` حساب جديد، `?` مساعدة الاختصارات.
- **بطاقة «مستحق اليوم»** في الرئيسية مع زر عرض الكل.
- **ميزانية شهرية** اختيارية مع شريط تقدّم في الرئيسية.
- **كشف التكرار** عند إضافة حساب مشابه (اسم / موقع / بريد).
- **حقول جديدة**: نهاية التجربة، إلغاء التجديد التلقائي، وسوم.
- **سجل تغييرات** السعر والاسم والعملة في صفحة التفاصيل.
- **المظهر**: فاتح / داكن / حسب النظام — يُحفظ في قاعدة البيانات.
- **التحقق من التحديثات** من GitHub في الإعدادات.
- **قاعدة بيانات v13**: أعمدة `trial_ends_on`، `renewal_cancelled`، جدول `subscription_audit_log`.

### Changed
- إغلاق النافذة على Linux لم يعد يُنهي التطبيق عند تفعيل الـ Tray.
